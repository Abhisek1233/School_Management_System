## 📖 Overview

The School Management System is a web-based MERN stack application designed to simplify school administration and improve communication between administrators, teachers, and students.

The platform provides role-based access for Admins, Teachers, and Students with features like attendance management, class organization, marks tracking, and performance monitoring.

---

## ✨ Features

- 🔐 Authentication & Authorization
- 👨‍🏫 Teacher Management
- 👨‍🎓 Student Management
- 🏫 Class & Subject Management
- 📅 Attendance Tracking
- 📊 Performance & Marks Management
- 📈 Dashboard & Data Visualization
- 📱 Responsive User Interface

## ⚙️ Installation & Setup

### 1️⃣ Clone Repository

```bash
git clone https://github.com/Abhisek1233/School_Management_System.git
```

---

### 2️⃣ Backend Setup

```bash
cd backend
npm install
npm start
```

Create a `.env` file inside the backend folder:

```env
MONGO_URL=your_mongodb_connection_string
```

Backend runs on:

```bash
http://localhost:5000
```

---

### 3️⃣ Frontend Setup

Open another terminal:

```bash
cd frontend
npm install
npm start
```

Frontend runs on:

```bash
http://localhost:3000
```
